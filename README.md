<h2>🛠️ Installation Steps:</h2>

<p>1. install git bash and run</p>

```
git clone https://github.com/Dextroxe/zekpto.git
```

<p>2. change directory</p>

```
cd zekpto
```

<p>3. install packages</p>

```
npm install
```

<p>4. finally run react using</p>

```
npm run dev
```
